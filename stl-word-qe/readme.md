# Prerequisites & Model Link

## Installation  

Install TransQuest using either of the following methods:  

### Install from source  
```bash
git clone https://github.com/TharinduDR/TransQuest.git
cd TransQuest
pip install .
```

### Or

Install using pip
```bash
pip install pip install transquest
```
